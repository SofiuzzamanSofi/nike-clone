## Cloning and Installing dependencies

First clone this project:

```bash
git clone https://github.com/SofiuzzamanSofi/nike-clone.git
```

Then install dependencies and modules files:

```bash
npm install
#or
npm i
#or
yarn
#or
pnpm
```
